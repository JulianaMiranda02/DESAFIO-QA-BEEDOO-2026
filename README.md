## Objetivo da Aplicação

O objetivo da aplicação é permitir que usuários **cadastrem cursos e visualizem uma lista com os cursos cadastrados**.

---

## Principais Fluxos da Aplicação

Os principais fluxos disponíveis no sistema são:

- **Cadastro de cursos** através de um formulário com as informações do curso.
- **Visualização da lista de cursos cadastrados**.
- **Exclusão de cursos cadastrados** da lista.

---

## Pontos Críticos para Teste

Os pontos mais críticos do sistema para teste são:

- **Formulário de cadastro de curso**, garantindo que todos os campos obrigatórios sejam preenchidos corretamente e que as validações funcionem.
- **Visualização da lista de cursos**, verificando se os cursos cadastrados aparecem corretamente.
- **Exclusão de cursos**, garantindo que o curso seja removido corretamente da lista.
